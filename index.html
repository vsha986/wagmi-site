What is WAGMI?

WAGMI is a meme-coin experiment built for fun and community. It’s a BEP-20 token on BNB Chain with straightforward tokenomics. No reflections, no tricky taxes—just simple transfers and optional burns by any holder.

  • Burnable by holders
  • No transfer taxes
  • Liquidity lock recommended at launch
  • Ownership can be renounced for trust (if using admin contract)

How to Buy

  1. Install MetaMask and switch network to BNB Chain.
  2. Fund your wallet with a small amount of BNB for gas.
  3. Open PancakeSwap and import the WAGMI contract address.
  4. Swap BNB/USDT → WAGMI. Keep slippage low (no taxes).
  5. Import the token in MetaMask to see your balance.
